SETTERS


Search Commands


  /  
Activate the search bar if it's not currently focused.
  @b | !b  
Show|Hide only benchmark problems.
  @t | !t  
Show|Hide only ticked problems.
  @y | !y $setter  
Show|Hide only problems set by $setter.
  @e  
Search setters instead of problems.
  @ | = $grade  
Show only problems matching $grade, e.g. @v5. 
  <$grade  
Show only problems no harder than $grade. 
  >$grade  
Show only problems at least as hard as the $grade. 
  @a | a= $ascents  
Show only problems with # of $ascents. 
  a<$ascents  
Show only problems with fewer than # $ascents. 
  a>$ascents  
Show only problems with at least # $ascents. 
  @s | s= $stars  
Show only problems with # of $stars. 
  s<$stars  
Show only problems with fewer than # $stars. 
  s>$stars  
Show only problems with at least # $stars.
